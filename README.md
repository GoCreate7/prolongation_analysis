# Анализ коэффициентов пролонгации

Тестовое задание для позиции Data Analyst / Аналитика данных.

## Описание задачи

Руководитель отдела сопровождения клиентов хочет оценить, насколько хорошо менеджеры пролонгируют договоры с клиентами. Для расчета используются два коэффициента пролонгации:

1. **Коэффициент 1** — пролонгация в первый месяц после завершения проекта
2. **Коэффициент 2** — пролонгация во второй месяц (для проектов, не пролонгированных в первый)

Необходимо:

1. Рассчитать коэффициенты пролонгации для каждого менеджера и для всего отдела в целом:
   - за каждый месяц
   - за 2023 год

2. Сформировать аналитический отчет в гугл-таблицах или excel.
